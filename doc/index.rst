Welcome to gpgliblib's documentation!
=====================================

**gpgliblib** is a Python library providing a common, well documented and
pythonic interface over various GPG libraries that exist for Python.

**gpgliblib** currently offers basic encryption and signing functions,
*basic* key handling features, GPG/MIME
(`RFC3156 <https://tools.ietf.org/html/rfc3156>`_) message handling and `Django
<https://www.djangoproject.com/>`_ integration.
